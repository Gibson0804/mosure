# 发布说明

## 版本历史

本文档记录Mosure Headless CMS (简称Mosure)的版本发布说明与变更记录。

---

## v1.0.0 (2026-01-25)

### 新增功能

#### 核心能力
- **低代码拖拽表单系统**：支持 20 种字段类型，可视化创建内容模型
- **AI 辅助建模**：使用自然语言描述需求，AI 自动生成表单结构
- **AI 内容生成**：支持单个和批量生成内容
- **MCP 服务**：内置 MCP Server，支持 AI 助手访问系统数据
- **OpenAPI 自动生成**：根据模型和函数自动生成 API 文档
- **任务化能力**：异步任务处理，支持进度查看和重试

#### 基础功能
- **项目管理**：多项目隔离，独立配置
- **内容管理**：完整的 CRUD 操作，支持发布/草稿状态
- **内容版本控制**：版本历史、对比、回滚
- **媒体资源管理**：文件上传、文件夹管理、标签管理
- **云函数**：Web 函数和触发函数，在线编辑和调试
- **环境变量**：项目级环境变量管理
- **定时任务**：基于 Cron 表达式的定时任务
- **触发器**：自动化规则引擎
- **插件系统**：可扩展的插件架构
- **审计日志**：操作记录和追踪
- **菜单管理**：自定义管理后台菜单
- **项目导入导出**：项目备份和迁移
- **API 管理**：API 密钥管理和调用监控

#### 开发者功能
- **开放 API**：完整的 RESTful API
- **API 文档**：自动生成的 OpenAPI 文档
- **插件开发**：完整的插件开发框架

### 技术栈

- **后端**：PHP 8.2+, Laravel
- **前端**：React, TypeScript, Vite
- **数据库**：SQLite (开发), MySQL (生产)
- **队列**：database / sync
- **存储**：本地存储 / 云存储
