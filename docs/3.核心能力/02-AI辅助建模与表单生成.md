# AI 辅助内容与表单生成

## 什么是 AI 辅助建模？

Mosure 集成了 AI 能力，可以帮助你快速创建内容模型和生成内容。无论你是想设计一个复杂的表单结构，还是批量生成文章内容，AI 都能提供智能辅助，大大提高工作效率。

### 核心功能

- **智能建模**：用自然语言描述你的需求，AI 自动生成表单结构
- **内容生成**：基于模型定义和提示词，自动生成内容
- **批量处理**：支持批量生成内容，适合快速填充数据
- **异步任务**：长时间任务通过任务中心管理，不阻塞操作

## AI 辅助建模

### 使用场景

当你不确定需要哪些字段，或者想快速创建一个模型时，可以让 AI 帮你设计：

- 创建博客文章模型
- 设计产品信息表单
- 构建活动报名表单
- 其他任何内容模型

### 如何使用

1. 进入"内容模型"页面
2. 点击"AI 建议"按钮
3. 在弹出的对话框中，用自然语言描述你的需求

### 提示词示例

**简单描述**：
```
创建一个博客文章模型，包含标题、内容、分类、发布时间、封面图
```

**详细描述**：
```
创建一个产品信息模型，需要以下字段：
- 产品名称（单行文本，必填）
- 产品编号（单行文本）
- 价格（数字输入，必填）
- 库存（数字输入）
- 规格（下拉选择，选项：S/M/L/XL）
- 详情（富文本）
- 产品图（图片上传，支持多图）
- 上架时间（日期时间）
- 是否上架（开关）
```

**场景描述**：
```
我需要创建一个活动报名表单，用于收集用户信息，包括姓名、手机号、邮箱、性别、年龄、报名时间和备注
```

### AI 生成结果

AI 会根据你的描述生成一个完整的表单结构，包括：

- 字段类型（单行文本、富文本、下拉选择等）
- 字段标签和字段名
- 是否必填的设置
- 选项列表（对于选择类字段）

你可以在 AI 生成的基础上进行调整，添加、修改或删除字段。

## AI 内容生成

### 单个内容生成

当你创建了一个模型后，可以让 AI 自动生成内容：

1. 进入"内容管理"页面
2. 选择一个模型
3. 点击"AI 生成"按钮
4. 输入生成提示词，描述你想要的内容

**提示词示例**：
```
写一篇关于人工智能发展趋势的文章，包含引言、三个主要观点和结论
```

**提示词示例**：
```
生成一个产品信息，产品名称为"智能手表"，价格为999元，库存100件
```

### 批量内容生成

如果你需要快速填充大量内容，可以使用批量生成功能：

1. 进入"内容管理"页面
2. 选择一个模型
3. 点击"批量生成"按钮
4. 设置生成数量和提示词模板

**提示词模板示例**：
```
写一篇关于{主题}的文章，字数500字左右
```

系统会根据模板生成多个内容，你可以指定不同的主题列表。

### 异步任务处理

批量生成可能需要较长时间，系统会创建一个异步任务：

1. 任务会自动加入任务队列
2. 你可以在"任务中心"查看任务进度
3. 任务完成后，内容会自动保存到数据库
4. 你可以随时查看任务日志和结果

## AI 配置

### AI 服务提供商

Mosure 支持多种 AI 服务提供商，你可以在系统配置中选择：

- 智谱AI
- deepseek
- 百度千帆
- 阿里千问
- 腾讯混元

### API 密钥配置

在使用 AI 功能前，需要配置 AI 服务的 API 密钥：

1. 进入"系统配置/system-config" → "大模型提供商"
2. 选择 AI 服务提供商
3. 输入 API 密钥
4. 保存配置

> ⚠️ **注意**：API 密钥会加密存储，请妥善保管。

## 使用技巧

### 建模技巧

- **明确需求**：描述越详细，AI 生成的结果越准确
- **参考示例**：如果不确定如何描述，可以参考已有的模型结构

### 内容生成技巧

- **指定格式**：在提示词中明确指定内容的格式和结构
- **提供上下文**：告诉 AI 生成内容的背景和目标受众
- **控制长度**：指定字数或段落数量，避免内容过长或过短
- **多次迭代**：如果第一次生成的结果不理想，可以调整提示词重新生成

### 批量生成技巧

- **使用变量**：在提示词模板中使用变量，让 AI 根据不同变量生成不同内容
- **合理设置数量**：根据实际需求设置生成数量，避免生成过多无用内容
- **检查结果**：批量生成后，建议抽查部分内容，确保质量符合预期

## 常见问题

### AI 生成的模型可以修改吗？

可以。AI 生成只是一个起点，你可以在表单构建器中自由调整字段，添加、删除或修改任何字段。

### AI 生成的内容准确吗？

AI 生成的内容基于提示词和模型定义，质量取决于提示词的清晰度和 AI 服务的能力。建议在使用前检查和编辑生成的内容。

### 批量生成需要多长时间？

批量生成的时间取决于生成数量、内容复杂度和 AI 服务的响应速度。通常几个到几十个内容需要几分钟到几十分钟不等。

### 如何查看 AI 生成的任务进度？

进入"任务中心"页面，可以看到所有正在运行和已完成的任务，包括任务状态、进度、日志等信息。

### AI 功能会产生额外费用吗？

AI 功能需要调用第三方 AI 服务，可能会产生 API 调用费用。具体费用取决于你选择的 AI 服务提供商和使用量。建议在使用前了解清楚计费规则。

## 最佳实践

1. **从简单开始**：先用 AI 生成一个基础模型，再根据实际需求调整
2. **迭代优化**：不断调整提示词，找到最适合你需求的描述方式
3. **人工审核**：AI 生成的内容建议人工审核后再发布
4. **合理使用**：AI 是辅助工具，不能完全替代人工创作
5. **控制成本**：合理设置生成数量，避免不必要的 API 调用
