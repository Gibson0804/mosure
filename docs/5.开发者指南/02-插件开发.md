# 插件开发

## 插件概述

Mosure 插件系统允许开发者扩展系统功能，添加自定义的内容模型、云函数、菜单项等，而无需修改核心代码。
可以查看官方提供的todo插件快速上手。

### 核心概念

- **PluginInterface**：插件必须实现的接口
- **AbstractPlugin**：提供默认实现的抽象基类
- **PluginManager**：插件管理器，负责发现、安装、卸载等
- **生命周期钩子**：插件在不同阶段可以执行的回调

## 插件目录结构

```
plugin-name
├── plugin.json          # 插件配置文件
├── PluginName.php       # 插件主类文件
├── functions/           # 云函数（可选）
│   ├── endpoints/       # Web 函数目录
│   │   ├── function1.json
│   │   └── function2.json
│   ├── hooks/           # 触发函数目录
│   │   ├── hook1.json
│   │   └── hook2.json
│   ├── variables.json   # 变量定义（可选）
│   ├── triggers.json    # 触发器配置（可选）
│   └── schedules.json   # 定时任务（可选）
├── menus/               # 菜单配置（可选）
│   ├── menu1.json
│   └── menu2.json
├── models/              # 内容模型（可选）
│   ├── model1.json
│   └── model2.json
├── data/                # 内容模型数据（可选）
│   └── model1/
│       ├── content1.json
│       └── content2.json
├── src/                 # PHP 源代码文件（可选）
│   └── CustomHandler.php
└── frontend/            # 前端资源（可选）
    ├── index.html
    └── assets/
```

## 创建插件

### 1. 创建插件目录

```bash
mkdir -p Plugins/MyPlugin
cd Plugins/MyPlugin
```

### 2. 创建插件主类

```php
<?php

namespace Plugins\MyPlugin;

use Plugins\AbstractPlugin;

class MyPlugin extends AbstractPlugin
{
    public function getName(): string
    {
        return 'My Plugin';
    }

    public function getVersion(): string
    {
        return '1.0.0';
    }

    public function getDescription(): string
    {
        return 'My custom plugin for Mosure';
    }

    public function install(): bool
    {
        // 安装逻辑
        return true;
    }

    public function uninstall(): bool
    {
        // 卸载逻辑
        return true;
    }
}
```

### 3. 创建配置文件

#### models/article.json

```json
{
  "name": "article",
  "title": "文章",
  "table_name": "model_articles",
  "mold_type": "list",
  "fields": [
    {
      "name": "title",
      "type": "input",
      "label": "标题",
      "required": true
    },
    {
      "name": "content",
      "type": "textarea",
      "label": "内容",
      "required": true
    }
  ],
  "settings": {},
  "list_show_fields": ["title"]
}
```

#### functions/endpoints/hello.json

```json
{
  "name": "hello",
  "slug": "hello",
  "type": "endpoint",
  "description": "Hello World",
  "runtime": "php",
  "code": "<?php\nreturn ['code' => 200, 'data' => ['message' => 'Hello']];",
  "http_method": "POST",
  "timeout_ms": 30000,
  "max_mem_mb": 128
}
```

#### functions/hooks/on_create.json

```json
{
  "name": "on_content_created",
  "slug": "on_content_created",
  "type": "hook",
  "description": "内容创建时触发",
  "runtime": "php",
  "code": "<?php\nreturn ['code' => 200, 'data' => ['processed' => true]];"
}
```

#### functions/variables.json

```json
[
  {
    "name": "plugin_api_key",
    "value": "",
    "remark": "插件 API 密钥"
  }
]
```

## 插件功能

### 内容模型

插件可以定义自定义内容模型，每个模型一个 JSON 文件：

```json
{
  "name": "article",
  "title": "文章",
  "table_name": "model_articles",
  "mold_type": "list",
  "fields": [...],
  "settings": {},
  "list_show_fields": ["title"]
}
```

### 云函数

插件可以提供 Web 函数和触发函数，分别放在 `functions/endpoints/` 和 `functions/hooks/` 目录：

**Web 函数示例** (`functions/endpoints/hello.json`)：
```json
{
  "name": "hello",
  "slug": "hello",
  "type": "endpoint",
  "description": "Hello World",
  "runtime": "php",
  "code": "<?php return ['code' => 200, 'data' => ['message' => 'Hello']];",
  "http_method": "POST",
  "timeout_ms": 30000,
  "max_mem_mb": 128
}
```

**触发函数示例** (`functions/hooks/on_create.json`)：
```json
{
  "name": "on_content_created",
  "slug": "on_content_created",
  "type": "hook",
  "description": "内容创建时触发",
  "runtime": "php",
  "code": "<?php return ['code' => 200, 'data' => ['processed' => true]];"
}
```

### 菜单

插件可以添加自定义菜单，每个菜单一个 JSON 文件：

```json
{
  "title": "博客管理",
  "icon": "blog",
  "target_type": "group",
  "order": 800,
  "visible": true,
  "children": [...]
}
```

### 环境变量

插件可以定义环境变量 (`functions/variables.json`)：

```json
[
  {
    "name": "plugin_api_key",
    "value": "",
    "remark": "插件 API 密钥"
  }
]
```

### 触发器

插件可以定义触发器，每个触发器一个 JSON 文件：

```json
{
  "name": "内容创建触发器",
  "event": "content.created",
  "action_function_slug": "on_content_created",
  "priority": 10
}
```

### 定时任务

插件可以定义定时任务，每个任务一个 JSON 文件：

```json
{
  "name": "每日备份",
  "cron": "0 0 * * *",
  "function_slug": "daily_backup",
  "enabled": true
}
```

### 数据导入

插件可以提供初始数据 (`data/` 目录)：

```json
[
  {
    "title": "示例文章",
    "content": "这是示例内容"
  }
]
```

### PHP 源代码

插件可以包含 PHP 源代码文件 (`src/` 目录)，用于扩展系统功能。

### 前端文件

插件可以包含前端文件 (`frontend/` 目录)，用于自定义前端界面。

## 生命周期钩子

### 安装钩子

```php
public function onBeforeInstall(): void
{
    // 安装前执行
}

public function onAfterInstall(): bool
{
    // 安装后执行
    return true;
}
```

### 卸载钩子

```php
public function onBeforeUninstall(): void
{
    // 卸载前执行
}

public function onAfterUninstall(): bool
{
    // 卸载后执行
    return true;
}
```

## 插件 API

### 注册路由

```php
public function registerRoutes(): void
{
    Route::prefix('my-plugin')
        ->group(function () {
            Route::get('/', [MyController::class, 'index']);
        });
}
```

### 注册监听器

```php
public function registerListeners(): void
{
    Event::listen(ContentCreated::class, function ($event) {
        // 处理事件
    });
}
```

## 最佳实践

- **最小侵入**：插件不应修改核心代码
- **事务保护**：数据库操作使用事务
- **错误处理**：完善的异常捕获和日志
- **文档完善**：提供清晰的 README
- **版本兼容**：明确支持的 Mosure 版本
