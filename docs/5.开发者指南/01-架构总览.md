# 架构总览

## 技术栈

Mosure 采用现代化的前后端分离架构：

### 后端

- **框架**：Laravel 11.x
- **语言**：PHP 8.2+
- **数据库**：SQLite (开发) / MySQL (生产)
- **队列**：database / sync
- **缓存**：file / database

### 前端

- **框架**：React 18
- **语言**：TypeScript
- **构建工具**：Vite
- **UI 框架**：Inertia.js
- **组件库**：Ant Design

## 核心模块

### 1. 项目管理

- **控制器**：`ProjectController`
- **服务**：`ProjectService`
- **模型**：`Project`
- **功能**：项目创建、选择、配置

### 2. 内容建模

- **控制器**：`MoldController`
- **服务**：`MoldService`
- **模型**：`Mold`, `MoldField`
- **功能**：内容模型管理、字段配置

### 3. 内容管理

- **控制器**：`ContentController`
- **服务**：`ContentService`
- **模型**：`Content`
- **功能**：内容 CRUD、版本控制

### 4. 媒体管理

- **控制器**：`MediaController`
- **服务**：`MediaService`
- **模型**：`Media`, `MediaFolder`, `MediaTag`
- **功能**：文件上传、文件夹管理

### 5. 云函数

- **控制器**：`CloudFunctionController`
- **服务**：`CloudFunctionService`
- **模型**：`CloudFunction`
- **功能**：Web 函数、触发函数

### 6. MCP 服务

- **控制器**：MCP 路由处理
- **服务**：MCP Server
- **模型**：MCP 相关
- **功能**：Tools 和 Resources

### 7. 插件系统

- **管理器**：`PluginManager`
- **接口**：`PluginInterface`
- **目录**：`Plugins/`
- **功能**：插件加载、卸载、生命周期

### 1. 内容创建流程

```

                    ↓
               创建数据表
```

### 2. API 调用流程

```

```

```

          (中间件)         (服务)
```

## 关键入口

### 路由注册

`bootstrap/app.php` 负责注册所有路由：

```php
// 基础路由
web: __DIR__.'/../routes/web.php',

// 额外路由
then: function () {
    // 前端页面路由
    Route::get('/frontend/{project_prefix}/{plugin}/{path?}', ...);

    // MCP 服务路由
    Route::prefix('mcp')
        ->middleware(\App\Http\Middleware\VerifyMcpAccess::class)
        ->group(function () {
            Mcp::web('/', \App\Mcp\Servers\mcpServer::class);
        });

    // 开放 API 路由
    Route::middleware(['api', \App\Http\Middleware\ValidateApiKey::class])
        ->prefix('open')
        ->group(base_path('routes/open.php'));

    // 客户端路由
    Route::middleware('client')
        ->prefix('client')
        ->group(base_path('routes/client.php'));

    // Chrome 扩展路由
    Route::middleware('client')
        ->prefix('api/chrome')
        ->group(base_path('routes/chrome.php'));
}
```

### 中间件

- **EnsureProjectSelected**：确保已选择项目
- **ValidateApiKey**：验证 API 密钥
- **VerifyMcpAccess**：验证 MCP 访问权限
- **LogRequestId**：记录请求日志
- **OpenCors**：处理 Open API 的 CORS

## 前后端交互

### Inertia.js

Mosure 使用 Inertia.js 实现前后端通信：

- **后端**：返回 Inertia 响应，包含页面数据和组件名称
- **前端**：Inertia 接收响应，渲染对应的 React 组件
- **数据流**：通过 props 和共享数据传递数据

### API 调用

前端通过以下方式调用后端 API：

- **页面路由**：Inertia 页面导航
- **Web API**：`/webapi/*` 路由，返回 JSON
- **Open API**：`/open/*` 路由，需要 API 密钥
- **Client API**：`/client/*` 路由，客户端专用
