# 菜单管理

## 什么是菜单管理？

菜单管理允许你自定义管理后台的左侧菜单，添加、编辑、删除菜单项，调整菜单结构。通过菜单管理，你可以根据项目需求定制管理界面，提升用户体验。

### 核心功能

- **菜单树管理**：可视化的菜单树结构
- **菜单项编辑**：添加、编辑、删除菜单项
- **菜单排序**：拖拽调整菜单顺序

---

## 菜单结构

菜单采用树形结构，支持多级嵌套：

```
管理后台
├── 仪表盘
├── 内容管理
│   ├── 内容列表
│   └── 内容模型
├── 云函数
│   ├── Web 函数
│   └── 触发函数
└── 系统设置
    ├── 项目配置
    └── 系统配置
```

## 创建菜单

用户可以自定义非核心菜单，便于把内容模型分组管理，比如同一个项目里有多种内容模型，比如文章相关模型，运营相关模型，可以创建分别一个文章菜单和一个运营菜单，然后将这些内容模型分别添加到对应一级菜单中。

### 基本步骤

1. 进入"菜单管理"页面
2. 点击"新建菜单"按钮
3. 填写菜单信息：
   - **菜单名称**：菜单的显示名称
   - **图标**：选择菜单图标
   - **选择包含的二级菜单**：选择包含的二级菜单（每个模型内容菜单只能属于一个一级菜单）
4. 点击"保存"完成创建

## 菜单管理

### 编辑菜单

1. 在菜单树中，点击菜单项的"编辑"按钮
2. 修改菜单信息
3. 点击"保存"

### 删除菜单

1. 在菜单树中，点击菜单项的"删除"按钮
2. 确认删除

> ⚠️ **警告**：删除父级菜单会同时删除所有子菜单。

### 移动菜单

1. 在菜单树中，拖拽菜单项到目标位置
2. 松开鼠标，菜单会自动移动

### 菜单排序规则

菜单按照以下规则排序：

1. **同级菜单**：按照 `order` 字段的值从小到大排序
2. **order 值**：数值越小，菜单越靠前显示
3. **默认值**：新建菜单的 `order` 默认为 0
4. **调整顺序**：可以通过以下方式调整菜单顺序：
   - 拖拽菜单项到目标位置（自动更新 order 值）
   - 编辑菜单的 `order` 字段手动设置顺序
   - 移动菜单到其他父级菜单下

### 系统核心菜单 order 值

系统核心菜单的 order 值如下，方便你控制自定义菜单的位置：

| 菜单名称 | key | order 值 |
|---------|-----|---------|
| 仪表盘 | dashboard | 100 |
| 模型管理 | mold | 200 |
| 内容列表 | content_list | 300 |
| 内容单页 | content_single | 400 |
| 媒体资源 | media | 500 |
| 云函数管理 | cloud_fn | 600 |
| 插件 | plugins | 700 |
| 项目管理 | manage | 900 |

> 💡 **提示**：
> - 如果你想让自定义菜单显示在仪表盘之前，设置 order < 100
> - 如果你想让自定义菜单显示在仪表盘和内容类型管理之间，设置 100 < order < 200
> - 如果你想让自定义菜单显示在所有核心菜单之后，设置 order > 900
> - 建议使用 10、20、30 这样的整数值，方便后续插入新菜单

### 启用/禁用菜单

1. 在菜单树中，点击菜单项的"启用/禁用"按钮
2. 确认操作

禁用菜单后，该菜单项将不会显示在左侧菜单中。

## 菜单图标

Mosure 使用 Ant Design Icons 作为默认图标库，提供了丰富的图标选择。

### 选择图标

1. 在菜单编辑页面，点击图标选择器
2. 搜索或浏览图标
3. 选择图标

> 💡 **提示**：图标字段为可选，如果不选择图标，菜单项将不显示图标。

---

## 常见问题

### 可以创建多少级菜单？

目前支持两级菜单：一级菜单（菜单组）和二级菜单（具体功能菜单）。


### 菜单顺序如何调整？

在菜单树中，拖拽菜单项到目标位置即可调整顺序。也可以通过编辑菜单的 `order` 字段来调整顺序。

### 菜单图标可以自定义吗？

目前系统使用 Ant Design Icons，不支持上传自定义 SVG 图标。
