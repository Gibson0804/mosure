# 云函数总览

## 什么是云函数？

云函数是 Mosure 提供的服务端代码执行能力，允许你编写自定义的业务逻辑，无需搭建服务器。云函数分为两种类型：Web 函数（HTTP Endpoint）和触发函数（Hook）。

### 核心功能

- **Web 函数**：通过 HTTP 接口对外提供服务
- **触发函数**：在特定事件发生时自动执行
- **在线编辑**：在管理后台直接编写和调试代码
- **执行记录**：查看函数的执行历史和日志
- **函数网关**：通过开放 API 调用函数

---


## 函数类型

### Web 函数

Web 函数是通过 HTTP 接口对外提供的自定义服务，类似于 API 接口，但完全由你编写代码实现。你可以通过函数网关调用 Web 函数，实现各种业务逻辑。

### 核心功能

- **HTTP 接口**：通过 HTTP 请求调用函数
- **自定义逻辑**：完全由你编写业务逻辑
- **参数传递**：支持 JSON 格式的请求参数
- **返回结果**：返回 JSON 格式的响应
- **执行记录**：查看每次调用的执行日志

**适用场景**：

- 数据处理和转换
- 第三方 API 集成
- 自定义业务逻辑
- 数据验证和校验

### 触发函数

触发函数是在特定事件发生时自动执行的函数，无需手动调用。你可以配置触发函数在内容创建、更新、删除等事件时自动执行，实现自动化业务逻辑。

### 核心功能

- **事件触发**：在特定事件发生时自动执行
- **在线编辑**：在管理后台直接编写代码
- **执行记录**：查看函数的执行历史和日志
- **本地测试**：在管理后台测试函数逻辑

---

**适用场景**：

- 内容创建/更新/删除时触发
- 定时任务
- 数据同步
- 通知发送

## 创建云函数

### 基本步骤

1. 进入"云函数"页面
2. 选择函数类型（Web 函数或触发函数）
3. 点击"新建函数"按钮
4. 填写函数信息：
   - **函数名称**：函数的显示名称
   - **函数标识**：函数的唯一标识（slug）
   - **描述**：函数的功能说明
5. 编写函数代码
6. 点击"保存"完成创建

---

## 函数管理

### 编辑函数

1. 在函数列表中，点击函数的"编辑"按钮
2. 修改函数代码或配置
3. 点击"保存"

### 启用/禁用函数

1. 在函数列表中，点击函数的"启用/禁用"按钮
2. 确认操作

禁用函数后，该函数将不再执行。

### 删除函数

1. 在函数列表中，点击函数的"删除"按钮
2. 确认删除

> ⚠️ **警告**：删除函数后，调用该函数的请求将失败。

### 查看执行记录

1. 在函数列表中，点击函数的"执行记录"按钮
2. 查看函数的执行历史，包括：
   - 执行时间
   - 执行状态
   - 执行结果
   - 错误信息（如果有）

## 函数网关

### 调用 Web 函数

Web 函数可以通过函数网关调用：

**接口地址**：
```
POST /open/func/{slug}
```

**请求头**：

```
X-API-Key: your_api_key_here
```

**请求体**：根据函数定义

**响应示例**：

```json
{
  "code": 200,
  "data": {
    "result": "函数执行结果"
  }
}
```

## 函数代码

### 编写函数代码

Mosure 支持使用 PHP 编写函数代码。函数代码在闭包上下文中执行，可以直接使用预定义的变量和工具类。

### 函数签名

```php
// 函数代码会自动包装在闭包中，直接编写逻辑即可
// 可用的变量：$payload, $env, $ctx, $Http, $db, $plugin
// 还包括：$request, $prefix, $event, $envs

// 处理逻辑
$result = [
    'code' => 200,
    'data' => '处理结果'
];

return $result;
```

### 可用变量

- **`$payload`**：函数输入参数（请求体）
- **`$env`**：项目环境变量（从 FunctionEnv 表读取）
- **`$ctx`**：执行上下文，包含：
  - `prefix`：项目前缀
  - `request`：HTTP 请求对象（Web 函数）
  - `event`：触发事件（触发函数）
  - `timeout_ms`：超时时间（毫秒）
  - `max_mem_mb`：内存限制（MB）
- **`$request`**：HTTP 请求对象（Web 函数专用）
- **`$prefix`**：项目前缀
- **`$event`**：触发事件名称（触发函数专用）
- **`$envs`**：环境变量数组

### 工具类

#### HTTP 请求（`$Http`）

```php
// GET 请求
$response = $Http->get('https://api.example.com/data', ['key' => 'value']);

// POST 请求
$response = $Http->post('https://api.example.com/create', ['name' => 'test']);

// 自定义请求
$response = $Http->send('PUT', 'https://api.example.com/update', [
    'json' => ['field' => 'value']
], ['Authorization' => 'Bearer token']);
```

#### 数据库操作（`$db`）

```php
// 查询数据（最多 100 条）
$users = $db->select('{project_prefix}_users', ['status' => 'active'], ['id', 'name'], 100);

// 查询单条记录
$user = $db->first('{project_prefix}_users', ['id' => 1]);

// 插入数据
$db->insert('{project_prefix}_users', ['name' => 'John', 'email' => 'john@example.com']);

// 更新数据
$db->update('{project_prefix}_users', ['status' => 'inactive'], ['id' => 1]);

// 删除数据
$db->delete('{project_prefix}_users', ['id' => 1]);

// 计数
$count = $db->count('{project_prefix}_users', ['status' => 'active']);

// 获取查询构建器（用于复杂查询）
$query = $db->query('{project_prefix}_users');
$result = $query->where('status', 'active')->orderBy('created_at', 'desc')->get();
```

> ⚠️ **安全限制**：数据库操作只能访问项目级表（表名必须以项目前缀开头）

### 执行限制

- **执行超时**：默认 3 秒，可配置（最小 100ms）
- **内存限制**：默认 64MB，可配置（最小 16MB）
- **返回结果大小**：最大 100KB，超出会被截断

### 返回值规范

函数应返回数组，格式如下：

```php
// 成功
return [
    'code' => 200,
    'message' => 'success',
    'data' => $result
];

// 失败
return [
    'code' => 400,
    'message' => 'error message',
    'data' => null
];
```

如果返回非数组，会自动包装为 `['data' => $value]`。

### 示例代码

详可见代码编辑页面的 模板

#### 示例 1：数据处理

```php
// 处理输入数据
$name = $payload['name'] ?? 'Guest';
$timestamp = $ctx['prefix'] . '_' . time();

// 调用外部 API
$response = $Http->get('https://api.example.com/user', ['name' => $name]);

return [
    'code' => 200,
    'data' => [
        'name' => $name,
        'timestamp' => $timestamp,
        'external_data' => $response
    ]
];
```

#### 示例 2：数据库操作

```php
// 查询用户数据
$users = $db->select($prefix . '_users', ['status' => 'active']);

// 插入日志
$db->insert($prefix . '_logs', [
    'action' => 'function_executed',
    'timestamp' => date('Y-m-d H:i:s')
]);

return [
    'code' => 200,
    'data' => [
        'user_count' => count($users),
        'message' => 'success'
    ]
];
```

#### 示例 3：触发函数

```php
// 获取事件数据
$eventType = $event;
$contentId = $payload['id'] ?? null;

// 根据事件类型处理
if ($eventType === 'content.created') {
    $db->insert($prefix . '_notifications', [
        'type' => 'new_content',
        'content_id' => $contentId,
        'created_at' => now()
    ]);
}

return [
    'code' => 200,
    'data' => ['processed' => true]
];
```

## 常见问题

### 函数执行时间有限制吗？

是的。函数执行时间限制为 60 秒，超时会自动终止。

### 函数可以访问数据库吗？

可以。函数代码可以访问项目数据库，执行查询和更新操作。

### 函数可以调用其他函数吗？

可以。函数代码可以调用其他 Web 函数。

### 如何调试函数？

1. 在函数编辑器中编写代码
2. 点击"测试"按钮
3. 输入测试参数
4. 查看执行结果和日志

### 函数执行失败会重试吗？

不会。函数执行失败后不会自动重试，需要手动重试。
