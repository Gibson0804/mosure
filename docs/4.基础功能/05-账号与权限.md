# 账号与权限

## 什么是账号与权限？

账号与权限管理系统控制用户对 Mosure 的访问权限。管理端账号用于访问管理后台，客户端账号用于通过 API 访问项目数据。

### 核心功能

- **账号管理**：创建、编辑、删除用户账号
- **登录认证**：用户登录和登出
- **密码管理**：找回密码、修改密码
- **Token 鉴权**：客户端 API 的 Token 认证

---

## 管理端账号

### 登录

1. 访问管理后台登录页面：`/login`
2. 输入用户名和密码
3. 点击"登录"按钮

### 登出

1. 点击右上角的用户头像
2. 点击"登出"按钮

### 找回密码

如果你忘记了密码：

1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 系统会发送重置密码链接到邮箱
4. 点击链接，设置新密码

### 修改密码

1. 点击右上角的用户头像
2. 点击"修改密码"
3. 输入旧密码和新密码
4. 点击"保存"

### 个人资料

1. 点击右上角的用户头像
2. 点击"个人资料"
3. 修改个人信息
4. 点击"保存"

## 客户端账号（客户端项目开发中）

### 客户端登录

客户端账号用于通过 API 访问项目数据：

**接口地址**：
```
POST /client/auth/login
```

**请求体**：

```json
{
  "username": "your_username",
  "password": "your_password"
}
```

**响应示例**：

```json
{
  "code": 200,
  "data": {
    "token": "your_access_token",
    "user": {
      "id": 1,
      "username": "your_username"
    }
  }
}
```

### Token 使用

获取 Token 后，在后续的 API 请求中携带 Token：

**方式 1：Authorization Header**

```
Authorization: Bearer your_token_here
```

### 获取当前用户信息

**接口地址**：
```
GET /client/me
```

**请求头**：

```
Authorization: Bearer your_token_here
```

**响应示例**：

```json
{
  "code": 200,
  "data": {
    "id": 1,
    "username": "your_username",
    "email": "your_email@example.com"
  }
}
```
