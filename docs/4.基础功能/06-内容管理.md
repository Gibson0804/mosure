# 内容管理

## 什么是内容管理？

内容管理是 Mosure 的核心功能，允许你创建、编辑、删除、发布内容。每个内容模型（如文章、产品）都有自己的内容列表，你可以像管理数据库记录一样管理这些内容。

### 核心功能

- **增删改查**：创建、编辑、删除、查看内容
- **发布管理**：控制内容的发布状态
- **批量操作**：批量发布、批量删除
- **AI 生成**：使用 AI 自动生成内容
- **版本控制**：查看内容的历史版本和变更

---

## 内容列表

### 访问内容列表

1. 进入"内容管理"页面
2. 选择一个内容模型
3. 显示该模型的所有内容

### 列表功能

内容列表支持以下功能：

- **搜索**：通过关键词搜索内容
- **筛选**：按字段筛选内容
- **排序**：按字段排序
- **分页**：分页浏览内容
- **批量操作**：批量发布、批量删除

### 列表显示

列表显示以下信息：

- 内容的主要字段（如标题、创建时间、状态等）
- 操作按钮（编辑、删除、发布等）

---

## 创建内容

### 基本步骤

1. 进入"内容管理"页面
2. 选择一个内容模型
3. 点击"新建内容"按钮
4. 填写表单（根据模型定义的字段）
5. 点击"保存"完成创建

### 内容状态

内容创建后默认为"草稿"状态，可以点击"发布"按钮发布内容。

---

## 编辑内容

1. 在内容列表中，点击内容的"编辑"按钮
2. 修改内容信息
3. 点击"保存"完成编辑

> 💡 **提示**：编辑内容会自动创建新的版本，你可以在"内容版本"中查看历史版本。

---

## 删除内容

### 单个删除

1. 在内容列表中，点击内容的"删除"按钮
2. 确认删除

### 批量删除

1. 在内容列表中，勾选要删除的内容
2. 点击"批量删除"按钮
3. 确认删除

> ⚠️ **警告**：删除操作不可恢复，请谨慎操作。

---

## 发布管理

### 发布内容

发布内容后，内容会通过开放 API 对外可见：

1. 在内容列表中，点击内容的"发布"按钮
2. 内容状态变为"已发布"

### 取消发布

取消发布后，内容不会通过开放 API 对外可见：

1. 在内容列表中，点击内容的"取消发布"按钮
2. 内容状态变为"草稿"

### 批量发布

1. 在内容列表中，勾选要发布的内容
2. 点击"批量发布"按钮
3. 确认发布

---

## AI 内容生成

### 单条生成

使用 AI 自动生成一条内容：

1. 进入"内容管理"页面
2. 选择一个内容模型
3. 点击"AI 生成"按钮
4. 输入生成提示词
5. 点击"提交"
6. 系统会自动生成内容并保存

### 批量生成

使用 AI 批量生成多条内容：

1. 进入"内容管理"页面
2. 选择一个内容模型
3. 点击"批量生成"按钮
4. 设置生成数量和提示词模板
5. 点击"提交"
6. 系统会创建一个异步任务
7. 在"任务中心"查看任务进度

详见《AI 辅助建模与表单生成》文档。

## 内容版本

Mosure 支持内容版本控制，你可以查看内容的历史版本和变更：

### 查看版本

1. 进入内容编辑页面
2. 点击"版本历史"按钮
3. 查看所有历史版本

### 版本对比

1. 在版本列表中，选择两个版本
2. 点击"对比"按钮
3. 查看两个版本的差异

### 版本回滚

1. 在版本列表中，选择要回滚的版本
2. 点击"回滚"按钮
3. 确认回滚

## 开放 API

Mosure 会根据内容模型自动生成开放 API，允许外部系统访问内容：

### 内容列表 API

```
GET /open/content/list/{table_name}
```

### 内容详情 API

```
GET /open/content/detail/{table_name}/{id}
```

### 创建内容 API

```
POST /open/content/create/{table_name}
```

### 更新内容 API

```
PUT /open/content/update/{table_name}/{id}
```

### 删除内容 API

```
DELETE /open/content/delete/{table_name}/{id}
```

详见《OpenAPI 与接口自动生成》文档。

## 常见问题

### 内容可以恢复吗？

删除的内容无法恢复，建议在删除前备份重要内容。

### 发布的内容可以被修改吗？

可以。已发布的内容可以继续编辑，编辑后需要重新发布才能让修改生效。

### AI 生成的内容准确吗？

AI 生成的内容基于提示词和模型定义，质量取决于提示词的清晰度和 AI 服务的能力。建议在使用前检查和编辑生成的内容。

### 如何批量操作内容？

在内容列表中勾选多个内容，可以进行批量发布、批量删除等操作。

### 内容版本会占用大量空间吗？

内容版本会占用一定的数据库空间，建议定期清理不需要的历史版本。
