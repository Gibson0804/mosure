# 系统配置

## 什么是系统配置？

系统配置是全局级别的设置，控制整个系统的行为和功能，不依赖于具体项目。系统配置包括邮件设置、AI 服务配置、存储配置等。

### 核心功能

- **全局配置**：查看和修改系统配置
- **服务配置**：配置邮件、AI 等第三方服务
- **配置测试**：测试配置是否正确
- **安全设置**：配置系统安全相关选项

---

## 访问系统配置

### 基本步骤

1. 点击左上角的系统菜单
2. 点击"系统配置"
3. 查看和修改配置

---

## 配置项

### 邮件配置

重置密码和云函数中可以使用该配置发邮件

- **邮件驱动**：选择邮件服务提供商（SMTP、Mailgun、SendGrid 等）
- **SMTP 主机**：SMTP 服务器地址
- **SMTP 端口**：SMTP 服务器端口
- **用户名**：SMTP 用户名
- **密码**：SMTP 密码
- **发件人地址**：默认发件人邮箱
- **发件人名称**：默认发件人名称

### AI 服务配置

- **AI 提供商**：选择 AI 服务提供商（OpenAI、百度文心一言等）
- **API 密钥**：AI 服务的 API 密钥
- **模型选择**：选择使用的 AI 模型
- **API 地址**：自定义 API 地址（可选）

### 存储配置

- **存储驱动**：选择存储方式（目前仅支持本地）

### 安全配置

- **会话过期时间**：用户会话的有效时间
- **密码复杂度**：密码复杂度要求

## 常见问题

### 系统配置会影响所有项目吗？

是的。系统配置是全局的，会影响所有项目。

### 配置修改会立即生效吗？

大部分配置修改会立即生效，但某些配置（如存储配置）可能需要重启服务。
