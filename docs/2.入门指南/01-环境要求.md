# 环境要求

## 服务端

- PHP：>= 8.2（以 `composer.json` 为准；安装向导的最低版本检查为 8.0.0）
- 数据库：SQLite 或 MySQL（安装向导支持二选一）
- PHP 扩展（安装向导会检查）：PDO、JSON、Fileinfo、OpenSSL、Tokenizer、Mbstring、Ctype、XML、Zip

## 前端

- Node.js：用于前端资源构建与开发（版本要求以实际 `package.json`/构建工具为准）

## 其他

- Composer：用于安装 PHP 依赖
